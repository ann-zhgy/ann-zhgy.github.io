安装

¶Java 安装

  • elasticsearch 需要 Java 的环境，我们一般需要安装 JDK1.8 或以上版本，Java 的安装步骤不再展示（参考菜鸟教程）……

¶Elasticsearch 安装

¶ 官网下载安装包安装

  1. 访问官网下载安装包

    可以在 官网 直接下载，也可以选择 Elasticsearch 和 Kibana 版本下载，也可以在官方 github 下载源码

Kibana 是 Elastic 公司提供的免费且开放的用户界面，方便用户对 Elastic 进行各种操作

  1. 解压之后，在 bin 目录运行 elasticsearch.bat 即可

  2. 运行完成之后，访问 127.0.0.1:9200，出现如下界面表示运行成功

  3. ES 目录介绍（需要我们关注的有四个）

    目录 作用
    bin 存放了启动及设置 ES 的入口（可执行文件）
    config 存放了 ES 的配置的信息
    data 存放数据
    logs 存放了 ES 运行过程中产生的日志信息
    plugins ES 支持的各种插件存放在该目录中

¶docker-compose 安装

  1. 准备配置文件

    • elasticsearch.yml：可以参考下载下来的包中的 config 中的 elasticsearch.yml 文件
    • kibana.yml：同理，参考下载下来的包中的 config 中的 kibana.yml 文件

  2. 编写 docker-compose

  3. 启动，访问 http://ip:port 验证是否安装成功

    1. 启动前需要将配置文件复制到相对应的挂载目录 (volumes) 中，然后才可以成功启动容器
    2. 这个 docker-compose 是一个单一节点的案例

¶ik 分词器下载安装

从 github 选择相对应的版本下载，然后放在目录 plugins/ik 下面，重启 ES 就好了

Elasticsearch 配置文件解析

ES 的配置文件在 config 文件夹下，值得关注的有 elasticsearch.yml、jvm.options、log4j2.properties，但是我们需要关注的仅仅是 elasticsearch.yml，jvm.options 在 JVM 的笔记中会详细解释，log4j2.properties 里面存储了日志打印的信息，一般不用管，而且在 elasticsearch.yml 中同样可以进行日志的配置

¶elasticsearch.yml

  • 其中很多配置请访问官方文档

¶jvm.options

对于学习来说，这个配置文件中我们只需要关注两个参数即可

  • 这两个参数用于设置 JVM 的内存，两个值设置为一致大小可以防止内存抖动（一个很正常的现象），ES 学习时推荐设置 512M 就基本足够了
